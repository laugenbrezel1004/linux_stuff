# Benutzerverwaltung: Benutzer "olaf" erstellen

## Aufgabenstellung

Erstellen Sie einen neuen Benutzer namens **olaf** mit folgenden automatischen Konfigurationen:

1. **Alias Definition**: Der Benutzer soll automatisch den Alias `lll` für `ls -lha` erhalten
2. **Datei erstellen**: Im Home-Verzeichnis von olaf soll automatisch eine Datei namens **steuergeld** vorhanden sein
