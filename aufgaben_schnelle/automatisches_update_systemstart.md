# Automatisches System-Update beim Start

## Aufgabe

Erstellen Sie ein Skript, das beim Systemstart automatisch System-Updates durchführt. Das Skript soll sicherstellen, dass das Betriebssystem und installierte Pakete auf dem neuesten Stand sind.

## Anforderungen

### Basis-Funktionalität
- ✅ Automatische Ausführung beim Systemstart
- ✅ Durchführung von System-Updates
- ✅ Protokollierung der Update-Aktivitäten
