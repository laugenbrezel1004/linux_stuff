 # Shell-Shortcuts: Tastenkombinationen und Befehle für effizientes Arbeiten in der Shell

## Navigation und Verzeichniswechsel

| Befehl/Tastenkombination | Funktion |
|--------------------------|----------|
| `cd` oder `cd ~` | Wechselt ins Home-Verzeichnis |
| `cd -` | Wechselt zum vorherigen Verzeichnis |
| `pwd` | Zeigt den absoluten Pfad des aktuellen Verzeichnisses an |
| `~` (in Pfaden) | Repräsentiert das Home-Verzeichnis (z.B. `cd ~/Dokumente`) |

## Befehlszeilenbearbeitung

| Tastenkombination | Funktion |
|-------------------|----------|
| **Ctrl + a** | Cursor an den Zeilenanfang setzen |
| **Ctrl + e** | Cursor ans Zeilenende setzen |
| **Alt + B** | Cursor ein Wort nach links bewegen |
| **Alt + F** | Cursor ein Wort nach rechts bewegen |
| **Ctrl + u** | Vom Cursor bis zum Zeilenanfang löschen |
| **Ctrl + k** | Vom Cursor bis zum Zeilenende löschen |
| **Ctrl + w** | Wort links vom Cursor löschen |
| **Ctrl + y** | Zuletzt gelöschten Text einfügen (Yank) |
| **Tab** | Automatische Vervollständigung von Befehlen oder Pfaden |
| **Tab Tab** | Zeigt mögliche Vervollständigungen an |

## Befehlsabbruch und Sitzung

| Tastenkombination | Funktion |
|-------------------|----------|
| **Ctrl + c** | Aktuellen Befehl abbrechen |
| **Ctrl + d** | Shell-Session beenden (wenn keine Eingabe aktiv) |
| **Ctrl + l** oder `clear` | Bildschirm leeren |

## Befehlshistorie

| Befehl/Tastenkombination | Funktion |
|--------------------------|----------|
| `history` | Liste aller zuvor ausgeführten Befehle anzeigen |
| **Ctrl + r** | Interaktive Suche in der Befehlshistorie |
| `!!` | Letzten Befehl erneut ausführen |
| `!befehl` | Letzten Befehl ausführen, der mit "befehl" beginnt |
| `!n` | Befehl mit Nummer n aus der History ausführen |
| **Ctrl + p** oder **Pfeil ↑** | Rückwärts durch die Historie blättern |
| **Ctrl + n** oder **Pfeil ↓** | Vorwärts durch die Historie blättern |
