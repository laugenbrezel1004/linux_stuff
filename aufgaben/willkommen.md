# 🚀 Mission: Das "Profi-Terminal"
Dein Terminal ist aktuell ziemlich kahl und schweigsam. Das ändern wir jetzt! Dein Ziel ist es, die Shell so zu konfigurieren, dass sie dich bei jedem Start mit den wichtigsten Systeminfos begrüßt.

Deine Aufgabe

Erstelle eine automatische Begrüßung, die beim Öffnen eines neuen Terminals erscheint. Die Ausgabe soll so aussehen:

```bash
Hallo [DeinName]!
Du bist gerade auf Host: [Rechnername]
Heute ist: [Datum]
Das System läuft bereits seit: [Laufzeit]
```



<details>
<summary>💡 Hilfe benötigt?</summary>
In deinem Home-Verzeichnis gibt es versteckte Konfigurationsdateien, welche den Namen deiner Shell trägt und auf "rc" endet, vielleicht 
kannst du damit etwas anfangen...
</details>
